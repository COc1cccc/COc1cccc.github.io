3151. Special Array I

Solution

配列を左から右にスキャンします。隣接する要素のペアごとに、それらのパリティが同じであれば、その配列は特別な配列ではないため、Falseを返します。そうでなければ、その配列は特別な配列であるため、Trueを返します。

3154. Find Number of Ways to Reach the K-th Stair

Solution

関数dfs(i, j, jump)をデサイーします。これは、現在i番目のステップにあり、操作1をj回、操作2をjump回行った状態で、k番目のステップに到達する方法の数を表します。答えはdfs(1, 0, 0)です。

関数dfs(i, j, jump)の計算プロセスは以下の通りです。

の場合、2回連続して下がることはできないため、k番目のステップにはもう到達できないので、0を返します。
の場合、k番目のステップに到達したと解釈され、答えは1で初期化され、その後計算を続けます。
の場合、下がることができるため、再帰的にdfs(i - 1, 1, jump)を計算します。
再帰的にdfs(i + 2^jump, 0, jump + 1)を計算し、答えに加算します。
