Solution --- 416. Partition Equal Subset Sum

April 18, 2022 1513

Description

Given an integer array nums, return true if you can partition the array into two subsets such that the sum of the elements in both subsets is equal or false otherwise.

Examples

Example 1:

Input: nums = [1,5,11,5]
Output: true
Explanation: The array can be partitioned as [1, 5, 5] and [11].

Example 2:

Input: nums = [1,2,3,5]
Output: false
Explanation: The array cannot be partitioned into equal sum subsets.

Constraints:

1 <= nums.length <= 200
1 <= nums[i] <= 100

Solution

This problem is a variant of the 0/1 knapsack problem, which can be formulated as follows: ‘Given an array, determine whether there exists a subset whose sum equals a given target sum.’

There are two main aspects to consider: determining the target sum and defining the recurrence formula.

In this problem, the target sum is implicitly defined as half of the total sum of all numbers in the array.

The recurrence formula is dp[sum] = dp[sum - num], where dp[sum] indicates whether the target sum can be achieved by summing up numbers. In this case, we only need to initialize dp[0].
