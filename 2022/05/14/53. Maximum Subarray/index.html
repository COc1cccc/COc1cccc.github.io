Solution --- 53. Maximum Subarray

May 14, 2022 1392

Description

Given an integer array nums, find the subarray with the largest sum, and return its sum.

Examples

Example 1:
Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: The subarray [4,-1,2,1] has the largest sum 6.

Example 2:
Input: nums = [1]
Output: 1
Explanation: The subarray [1] has the largest sum 1.

Example 3:
Input: nums = [5,4,-1,7,8]
Output: 23
Explanation: The subarray [5,4,-1,7,8] has the largest sum 23.

Constraints:

1 <= nums.length <= 105
-104 <= nums[i] <= 104

Solution

By constructing an array of the same length as the original one, the DP array implies the maximum sum of the subarray ending at dp[i].

Therefore, the following state transition formulas apply:

  • dp[i] = dp[i - 1] + nums[i] when nums[i] >= 0.
  • dp[i] = nums[i] when nums[i] < 0.
    The maximum sum of the subarray, which is sought by the problem, corresponds to the maximum value of the DP array.
